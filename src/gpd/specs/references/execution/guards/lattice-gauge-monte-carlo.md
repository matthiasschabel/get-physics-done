# Lattice-Gauge Monte Carlo Execution Guard

Use for selected `lattice-gauge-monte-carlo` bundles.

- State gauge action, fermion discretization, scale-setting observable,
  boundary conditions, volume, lattice spacing, and target continuum/chiral
  limits before interpreting measurements.
- Track autocorrelation, topology, acceptance rate, and effective sample size;
  topology freezing is a blocker for topology-sensitive claims.
- For correlators, check effective-mass plateaus, excited-state contamination,
  covariance treatment, and operator normalization.
- For continuum or chiral claims, fit multiple lattice spacings or masses and
  record fit-family sensitivity.
- Compare against trusted lattice averages, exact small-volume tests, or known
  perturbative/asymptotic behavior before scaling production claims.
