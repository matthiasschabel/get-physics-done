# Domain Strategy Planning Guide Index

Use this index only when selected protocol bundles do not already provide
planner-specific `planning_guides`. Load the smallest matching guide set; do
not load the whole catalog.

Selection rule:

1. Choose the guide that controls the physics claim being planned.
2. Add a second guide only when the phase explicitly crosses a method boundary.
3. For cross-domain transfers, also load the convention bridge guide.
4. Keep the approved contract, anchors, forbidden proxies, user decisions, and
   proof obligations authoritative over every guide.

Guides:

- `references/planning/qft-perturbative-calculation.md` - amplitudes, Feynman diagrams, loops, counterterms, cross sections.
- `references/planning/condensed-matter-analytical.md` - Hamiltonians, order parameters, mean field, response, phase diagrams.
- `references/planning/condensed-matter-numerical.md` - ED, DMRG, QMC, DMFT, benchmarked production sweeps.
- `references/planning/statistical-mechanics.md` - ensembles, free energy, critical behavior, finite-size scaling.
- `references/planning/gr-cosmology.md` - spacetime background, gauges, perturbations, constraints, observables.
- `references/planning/amo-quantum-optics.md` - rotating frames, RWA, master equations, spectra, decoherence.
- `references/planning/numerical-pde-ode.md` - discretization, stability, convergence, production solves.
- `references/planning/effective-field-theory.md` - scale hierarchy, power counting, matching, running, truncation error.
- `references/planning/cross-domain-convention-bridge.md` - convention transfer when physics and method domains differ.
